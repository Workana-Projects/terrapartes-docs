# Quick Guide - Terrapartes

## Accessing the Editor

1. Log in to your Shopify Admin panel
2. Sidebar menu: **Online Store** > **Themes** > **Customize**

---

## Most Common Changes

### Change Site Colors
1. Click the gear icon (bottom left)
2. Look for **"Terrapartes"** section
3. Edit the desired colors
4. Save

### Add Supplier Logo
1. Find **"TP Marcas / Proveedores"** section
2. Click **"Add block"** > **"Marca / Proveedor"**
3. Enter supplier name
4. Click **"Select image"** > **"Upload"**
5. Select logo file (PNG recommended)
6. Save

### Change Category
1. Find **"TP Categorias"** section
2. Click on the category to edit
3. Change name, image and/or link
4. Save

### Edit Products in a Category
1. Go to **Shopify Admin** > **Pages** > create or select the page
2. Assign the **`page.category`** template
3. In the Theme Editor, navigate to that category page
4. Add **"Producto"** blocks with names and images
5. Save (each page saves its own products)

### Change WhatsApp Number
1. Click gear icon (bottom left)
2. Find **"Terrapartes"** > **"WhatsApp"**
3. Edit the number
4. Save

### Change Topbar Color
1. Click on **"TP Barra Superior"** section
2. Edit **"Color de fondo"** (Background color)
3. Save

### Change Footer Color
1. Click on **"TP Footer"** section
2. Edit **"Color de fondo"** (Background color)
3. Save

---

## Important Shortcuts

| Action | How to do it |
|--------|--------------|
| **Save** | Green button top right |
| **Undo** | Click "Discard" before saving |
| **Mobile view** | Device icons at top |
| **Hide section** | Eye icon next to section |
| **Reorder** | Drag sections/blocks |

---

## Recommended Image Formats

| Type | Format | Size |
|------|--------|------|
| Supplier logos | Transparent PNG | 200x100 px |
| Category images | JPG | 600x450 px |
| Hero image | JPG | 1920x800 px |
| Site logo | Transparent PNG | 300x150 px |

**Compress before uploading:** [tinypng.com](https://tinypng.com)

---

## Editable Sections

- **TP Barra Superior** - Blue bar at top
- **TP Header** - Logo and menu
- **TP Hero** - Large home image
- **TP Categorias** - Category cards
- **TP Marcas / Proveedores** - Supplier logos
- **TP Footer** - Page footer
- **Theme settings** - Global colors, WhatsApp, contact
